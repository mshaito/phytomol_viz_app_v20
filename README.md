# PhytoMol-Viz v2.0

Interactive Streamlit prototype for natural product computational drug discovery visualization.

## Features
- GC-MS Explorer
- Cytotoxicity dose-response visualization
- ADMET dashboard
- Target prediction tables
- Network pharmacology and PPI analysis
- Docking score comparison with a lightweight 3D placeholder viewer
- Molecular dynamics metrics, RMSD, RMSF, Rg, H-bonds, SASA
- Report summary and downloadable text
- Synthetic demo data plus CSV upload options

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Data notice
All included data are synthetic demonstration data and should not be interpreted as experimental or validated biological results.
