# Asset Inventory

This folder contains synthetic/demo visual assets for the PhytoMol-Viz prototype.

## icons
SVG icons for GC-MS, cytotoxicity, ADMET, target prediction, network pharmacology, PPI, docking, MD simulation, and reports.

## images
General workflow banner and placeholder images.

## protein_images
Placeholder protein images for AKT1, STAT3, PTGS2, and ESR1. These are not real structures.

## compound_images
Placeholder compound images for selected vetiver compounds. These are not real molecular structures.
